نظام الاستعلام برقم الهوية
===========================

المدرسة: عماد الدين زنكي المتوسطة
عدد السجلات: 192

طريقة التشغيل:
1) افتح ملف index.html مباشرة.
2) أدخل رقم الهوية أو الإقامة.
3) ستظهر بيانات الطالب فورًا.
4) يمكن طباعة النتيجة من زر "طباعة النتيجة".

الملفات:
- index.html: صفحة الاستعلام.
- students.json: ملف البيانات بصيغة JSON.
- students-data.js: نفس البيانات بصيغة JavaScript حتى تعمل الصفحة محليًا بدون سيرفر.

تنبيه:
البيانات تحتوي على أسماء وأرقام هويات، لذلك لا ترفعها على موقع عام إلا بعد إضافة تسجيل دخول وصلاحيات وصول.
